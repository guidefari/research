# Terraform Associate Exam Prep — Follow-Along Tutorial

A hands-on, follow-along tutorial series covering every topic on the **HashiCorp Terraform Associate (004)** exam. Work through each module in order, mark topics complete as you go, and come away ready to sit the exam with confidence.

> **Exam at a glance:** 1-hour online proctored exam · multiple-choice · $70.50 USD · valid 2 years
>
> Official exam page: https://developer.hashicorp.com/certifications/infrastructure-automation

---

## How to Use This Guide

1. Work through each file **in order** — they build on each other.
2. Every file has hands-on code you should **type out yourself**, not just copy-paste.
3. After finishing a file, come back here and **check the box** by editing this file and changing `[ ]` to `[x]`.
4. Links to official docs and further reading are included at the bottom of every file.

## Prerequisites

- A terminal / shell (bash, zsh, PowerShell)
- A free AWS account *or* you can use `local` provider examples (noted per topic)
- A text editor — VS Code with the [HashiCorp Terraform extension](https://marketplace.visualstudio.com/items?itemName=HashiCorp.terraform) is recommended
